Alter Source

Overview

Alter Source has a pretty simple mechanic. It's a sandbox mod, just like every else. BUT! It's not like every else. This one is different. And to find it out, play it.

Features

  • Lua Support: Write addons and have fun!
  • Better Physgun: Tired of using the old HL2 Beta physgun? Well uh, i think you'll like this mod then!
  • Awesomium Support: Support for Awesomium, for web stuff.
  • Updates: Quick updates, and even quicker changelogs

Installation

  1. Download the latest version of Alter Source from ModDB
  2. Extract the files to your sourcemods folder (ex.: C:/Steam/SteamApps/sourcemods)
  3. Restart Steam and enjoy!
